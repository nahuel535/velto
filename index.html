FAQ

Preguntas frecuentes.

¿Funciona desde el celular?

Sí. Velto es una app web responsiva — funciona desde cualquier dispositivo con navegador: celular, tablet o computadora. No requiere instalación ni descarga.

¿Necesito saber algo técnico para usarlo?

No. El sistema está diseñado para equipos de ventas, no para técnicos. El onboarding es asistido y las primeras semanas tenés soporte directo.

¿Cómo migro mi stock actual?

Hacemos el proceso juntos. Si tenés los datos en Excel o en papel, los cargamos durante el setup inicial. El formulario de alta guiado está pensado para que sea rápido y sin errores.

¿Se adapta a mi marca o es genérico?

Velto se personaliza completamente a tu tienda: nombre, colores, logo y dominio propios. XYLO es un ejemplo de implementación, no el modelo único. Cada cliente tiene su versión.

¿Qué pasa después del primer mes?

El sistema queda instalado y operativo. El soporte continúa según el plan acordado. Las actualizaciones y mejoras se suman en función del uso real del negocio.
